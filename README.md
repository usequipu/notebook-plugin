# notebook-plugin

Jupyter Notebook plugin for [Quipu](https://quipu.cc). View and execute `.ipynb` notebooks with live kernel support.

## Install

Open the Plugin Manager panel in Quipu → Available → Install.

## Supported file types

`.ipynb` (nbformat 3 and 4)

## Keybindings

| Command | Mac | Windows/Linux |
|---------|-----|---------------|
| Run All Cells | `⌘⇧↩` | `Ctrl+Shift+Enter` |
| Interrupt Kernel | `⌘⇧I` | `Ctrl+Shift+I` |
| Restart Kernel | `⌘⇧0` | `Ctrl+Shift+0` |

## Build from source

```bash
npm install
npm run build
```
